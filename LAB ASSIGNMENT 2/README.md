
# GradeBook Analyzer

A CLI tool to analyze and report student grades for the course "Programming for Problem Solving using Python".

## Features
- Manual entry or CSV import of student marks
- Statistical calculations (mean, median, min, max)
- Letter grade assignment (A-F)
- Pass/Fail lists using list comprehensions
- Formatted results table
- Optional CSV export of final grade table

## Quick start
1. Ensure Python 3.8+ is installed.
2. Run the script:
   ```
   python gradebook.py
   ```
3. Use the menu to choose manual entry, load a CSV (`sample_marks.csv` is provided), or run the demo data.
